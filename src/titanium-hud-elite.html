Deadman heartbeat (Supabase)

Heartbeat uses apikey + Authorization: Bearer (session JWT when signed in, or anon key if you enable anon-only below — matches curl smoke tests). Default path /functions/v1/rapid-endpoint. Paste URL/key in Settings only — never commit keys.

Emergency rescue: HUD POSTs { type: "trigger_rescue", contacts, location, route, timestamp } to the Deadman function URL (rapid-endpoint). Implement handling in that Edge Function.

Account: use 🔐 SIGN IN in the top toolbar (email, password; display name on sign-up). Session is stored by Supabase in this browser.
